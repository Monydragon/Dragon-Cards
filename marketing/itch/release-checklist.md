# itch.io Release Checklist

- Upload a Windows zip containing the published desktop build.
- Use `assets/cover-630x500.png` as the cover image.
- Use `assets/banner-1000x300.png` as the page banner or background art.
- Upload 3-5 screenshots from `artifacts/render-captures/`.
- Copy the text from `page-copy.md` into the project description.
- Use the metadata in `metadata.json`; do not mark untested platforms.
- Keep the page private or restricted until the build, cover, screenshots, and metadata are complete.
- Mention that this is a prototype and ask for feedback on clarity, balance, modes, and LAN play.
