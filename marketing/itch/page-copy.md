# Dragon Cards

**Short description:** A fantasy digital TCG prototype with elemental energy, deck building, boosters, tutorials, direct LAN play, and new prototype play modes.

## Description

Dragon Cards is a Windows desktop card-battle prototype built for fast playtesting. Build an elemental deck, manage persistent energy, sacrifice cards for tempo, block incoming attacks, open booster packs, level a local profile, and try multiple experimental ways to play.

This release is a prototype. It is meant to get the game into more hands for feedback on clarity, pacing, card balance, deck building, game modes, and the feel of combat.

## Features

- 480-card library across Core, Elemental Ascension, and Primal Clash cards
- Eight mono-element starter decks
- Local profile with name, level, Coins, owned cards, boosters, and starter unlocks
- Standard Dragon Duel matches against AI
- Local hotseat and direct LAN host/join multiplayer
- Mode Select with Dragon Duel, Starter Clash, Dragon Avatar, Sealed Gauntlet, Sandbox Lab, and Tutorial Trials
- Dragon Avatar prototype mode with singleton identity deck building
- Store/Packs screen with boosters, starter decks, singles, duplicate conversion, and pack opening
- Tutorials with one-time coin rewards
- Victory/defeat result screens with progression rewards and battle spoils
- Original generated logo, rarity icons, and sound effects

## Controls

- Mouse: select cards, buttons, targets, blockers, and energy
- Drag cards from hand to play zones where available
- Keyboard/controller support is present for menus and core match flow
- Options include fullscreen/window size, sound volume, mute, and card hover zoom

## Install And Run

Download the Windows build zip, extract it, and run `DragonCards.Desktop.exe`.

If Windows SmartScreen warns about the prototype, choose “More info” only if you trust this downloaded build.

## Prototype Notes

Please test:

- Whether card details are readable
- Whether blocking, targeting, combat actions, and sacrifice previews make sense
- Whether starter decks feel distinct
- Which cards feel too weak, too strong, or confusing
- Whether the new modes make the game feel more replayable
- Direct LAN host/join if you have another player on the same network
